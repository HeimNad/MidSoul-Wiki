# 饰品集册

## 粒子效果

截止目前，粒子效果全部根据玩家等级逐步解锁。

正在隐身的玩家不会有粒子效果，游戏中灵魂和守卫者互相看不到彼此的粒子效果（仅限以下的装饰性粒子）。

| 名称     | 解锁需求            | 描述           | 所使用粒子                    |
| :------- | :------------------ | :------------- | :---------------------------- |
| 午夜繁星 | 游戏等级达到 Lv.10  | 繁星指引归路。 | electric_spark                |
| 氤氲灵气 | 游戏等级达到 Lv.25  | 灵气叩开门扉。 | glow                          |
| 霜凝冰痕 | 游戏等级达到 Lv.40  | 冰痕铭刻岁月。 | ominous_spawning              |
| 点灯施援 | 游戏等级达到 Lv.55  | 灵灯照亮迷途。 | wax_on                        |
| 灵迹随踪 | 游戏等级达到 Lv.70  | 灵迹留下行踪。 | end_rod                       |
| 绚彩粉墨 | 游戏等级达到 Lv.85  | 粉墨演绎风华。 | `dust{color:[1,1,1],scale:1}` |
| 万象环身 | 游戏等级达到 Lv.100 | 万象于你归一。 | witch                         |
| 桃起樱落 | 游戏等级达到 Lv.114 | 绯色迷花飘散。 | cherry_leaves                 |
| 双生魂焰 | 游戏等级达到 Lv.128 | 魂火灼穿彼岸。 | flame + soul_fire_flame       |

## 武器样式

需要注意的是，武器样式不会对武器效能造成任何影响：例如形如重锤的“破魂锤”不会有重击效果。

| 名称     | 解锁需求                                     | 描述                 | 所使用物件 |
| :------- | :------------------------------------------- | :------------------- | :--------- |
| 祛灵刃   | 开始游戏（初始解锁）                         | 以锐利之刃斩不净之物 | 金剑       |
| 祛灵刃 α | 进度点达到 100 点                            | 以璀璨之刃斩不净之物 | 钻石剑     |
| 祛灵刃 β | 进度点达到 250 点                            | 以历战之刃斩不净之物 | 下界合金剑 |
| 破魂锤   | 达成进度 “[???](./进度碑刻)”                 | 这一击，贯穿脑门！   | 重锤       |
| 散炽棍   | 达成进度 “[事去人亡二百载](./进度碑刻#处决)” | 手感火热……注意防水！ | 烈焰棒     |
